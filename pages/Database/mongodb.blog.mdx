## MongoDB Magic

MongoDB is a popular NoSQL database that stores data in flexible, JSON-like documents. Let's take a look at a simple MongoDB query to illustrate:

```mongodb
// Example MongoDB Query
db.users.find({ age: { $gt: 25 } });
```
